Web Front End

Ínicio das turmas: 23 de maio
Avenida das Nações Unidas, 12225 - São Paulo


Este curso de Desenvolvimento Front-End é projetado para fornecer uma introdução abrangente aos princípios, técnicas e tecnologias fundamentais utilizadas na criação de interfaces de usuário interativas e responsivas para aplicações web. Ao longo do curso, os participantes serão guiados desde os conceitos básicos de HTML, CSS e JavaScript até técnicas mais avançadas de design responsivo e desenvolvimento de aplicações web modernas.



Objetivos do Cursos

  • Compreender os fundamentos do desenvolvimento web front-end, incluindo HTML, CSS e JavaScript.
  • Dominar as técnicas de criação de layouts responsivos e acessíveis. Explorar frameworks e bibliotecas populares, como Bootstrap e React.js.
  • Aprender sobre otimização de desempenho e melhores práticas de desenvolvimento front-end.
  • Desenvolver habilidades práticas através de projetos práticos e exercícios hands-on.
  • Explorar frameworks e bibliotecas populares, como Bootstrap e React.js.

Duraçao Do Curso

O curso será ministrado ao longo de 12 semanas, com aulas semanais
de 2 horas cada, totalizando 24 horas de instrução. Conteúdo Programático:

    Introdução ao Desenvolvimento Web Front-End

  • Visão geraç do front-end e sua importâcia
  • Breve hitória da web e suas tecnologias.

    HTML (HyperText Markup Language)

  • Estrutura básica de uma página HTML.
  • Tags, elementos e atributos HTML.
  • Semântica em HTML5.
  • Formulários e elementos de entrada.

    CSS (Cascading Style Sheets)

  • Sintaxe básica e regras de estilo.
  • Seletores CSS e especificidade.
  • Layouts com Flexbox e Grid.
  • Estilização avançada com CSS3.

    JavaScript (JS) Básico

  • Conceitos fundamentais de programação.
  • Manipulação do DOM (Document Object Model).
  • Eventos e manipulação de eventos.
  • Introdução a funções e escopo.
